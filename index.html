UFO Sightings: Shape Trends Over Time

Data & Methodology

Data Source: National UFO Reporting Center (NUFORC) Database. Data last pulled/processed on July 17, 2024.

JSON Down-sampling: The ufo_shapes.json file was generated by prep.py from the full NUFORC dataset. To ensure a manageable file size (target ≤ 2MB) for web delivery, the following steps were taken:

  • Only sightings from 1940 onwards were included.
  • UFO shapes with fewer than 200 total reported sightings in this period were grouped into an "Other" category.
  • Data was aggregated to count the number of sightings for each shape (or "Other") per year.
  • The final JSON includes year, shape, count, and decade for each aggregated entry.

Statistical Note: A Chi-squared (χ²) test performed on the relationship between UFO shapes and decades (from 1940s to 2020s, using the categorized shapes) indicates that the distribution of reported UFO shapes significantly varies across decades (p < 0.001). This suggests that the types of shapes people report seeing have changed over time. (This test was performed during data preparation; the browser does not execute statistical tests).
